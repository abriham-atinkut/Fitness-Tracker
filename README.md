# Fitness Tracker

A responsive web application to log workouts, track fitness progress, and explore exercises using **React and WGER API**.

---

## Project Overview

The Fitness Tracker allowes users to record workouts, view their workouts history, and track progress over time. Built with _React_ and styled with _Tailwind CSS_, the project demonstrates frontend skills including:

- API integration
- State management
- Responsive UI design
- Data visualization

---

## Getting Started

1. _Clone the repository:_

```bash
git clone
cd fitness-tracker
```

2. _Install dependencies:_

```bash
git install
```

3. _Run the development server:_

```bash
npm run dev
```
